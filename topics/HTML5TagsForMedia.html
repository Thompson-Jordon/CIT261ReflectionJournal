HTML5 Tags for Media

Video

The video tag allows you to embed a video into your web page. This is a big improvement over needing to use plug-ins.

Canvas

The canvas tag allows you to draw graphics using JavaScript. Mozilla Developer has a fantastic tutorial that walks you through the basics and can give you some practice with the tool.
