JavaScript

Loops

Loops are used to repeat a piece of code until certain conditions are met.

while

A while loop will check the condition, and if the condition is not met it will continue to run the code inside the loop.

while(condition) {
//do something
}

Infinite Loop

An Infinite Loop is a loop that will continue to loop forever due to the fact that the loop will never meet the conditions of the loop.

let n = 1;
while (n > 0) {
console.log('Please Stop!');
n++;
}

do while

The do while will run the loop once before it checks the condition. If the condition is not met it will continue to loop.

do {
//do something
} while (condition)

for

A for loop allows the user to set an initialized value, a condition to be met, and what to do after if the condition is not met.

for (let i = 0; i < 10; i++) {
//do something
}

Conditional Statements

if

The if statement allows the code to check a single condition. If the condition is true it will execute the code in the if statement, otherwise it does nothing.

if (a > b) {
//do something
}

if...else

The if else block allows you to check if the code meets the if condition. If it does it will execute the code if block. If it is false it will execute the else block.

if (a == 1) {
//do something
} else {
//do something else
}

if...else if...else

You can also have many "else if" statements. This allows you to check many different possibilities. User must end with an else statement

if (name == 'Fred') {
console.log('Hi Fred!');
} else if (name == 'Jon') {
console.log('Hi Jon');
} else {
console.log('Who are you?');
}

switch

The switch statement allows you to check value agains several cases. If your value doesn't match any of the cases it will use the default case. You need to use "break;" on each of your cases or else the code will keep executing the next case.

switch (color) {
case "blue":
console.log ("Your color is blue");
break;
case "red":
console.log ("Your color is red");
break;
default:
console.log("Your color is not red or blue");
}

Functions

A function is a self contained piece of code that you can reference by name. They are useful for reducing redundant code.

Declaring a Function

To declare a function you use the word "function" then give it a name. In this case the name is "greeting". You follow the name with "()" which are used for function arguments.

function greeting(){
console.log("Welcome");
}

Calling a Function

To call a function you simply enter its name as you can see in the following code snippet.

//Calling a function
greeting();
<< 'Welcome'

Inputs and Outputs

You can input values through a functions arguments. You can also recieve values from a function by using "return" in the function.

//A function to add 2 values
function sum(x, y){
return x + y;
}

/*When calling this function you must pass in arguments*/
sum(1, 2);
<< 3

Variables

let

In JavaScript instead of using "int" or "char" keywords, they use the keywords "let" to declare every type of variable.

let x = 3, name = 'Jane';

const

const is used to declare a constant variable. Once assigned you cannot change the value of the variable.

const PI = 3.14;
