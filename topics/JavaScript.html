Loops

Loops are used to repeat a piece of code until certain conditions are met.

while Loop

This will check the condition, and if the condition is not met it will continue to run the code inside the loop.

while(condition) {
//do something
}

Infinite Loop

An Infinite Loop is a loop that will continue to loop forever due to the fact that the loop will never meet the conditions of the loop.

let n = 1;
while (n > 0) {
console.log('Please Stop!');
n++;
}

do while Loop

This will run the loop once before it checks the condition. If the condition is not met it will continue to loop.

do {
//do something
} while (condition)

for Loop

A for loop allows the user to set an initialized value, a condition to be met, and what to do after if the condition is not met.

for (let i = 0; i < 10; i++) {
//do something
}
