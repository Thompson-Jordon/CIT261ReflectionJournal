Local Storage

Local Storage API

With local storage you can save data that can be used across sessions. This helps to make the app feel more responsive and robust while keeping any personal user data safely on the users computer. With local storage you don't need to send the data back to the server to store it. There are two kinds of storage "localStorage" and "sessionStorage". "localStorage" will hold the data long term and will be available even if the user closes the web page. "sessionStorage" allows the data to be stored only for the current session. As soon as the user closes the page the data is destroyed.

Storing and Retrieving Simple Data

When storing local data you use "key" : "value" pairs.
