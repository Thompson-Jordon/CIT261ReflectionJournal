JavaScript Objects

Object Creation Functions

To create an object in JavaScript you can either use curly braces or you can make your variable equal to a "new Object()"

//This uses curly braces
const rock = {};

//This uses 'new'
const bird = new Object();

You can also declare an object as a new class using the "class" keyword.

Notice how the "constructor()" keyword is used to initialize the values inside of the class.

Inheritance

Inheritance allows you to build off of the properties of one class to make another. To do this you use the "extends" keyword when creating a new class.

Properties

You can access your object properties with dot notation

//you can do it like this
bird.color
« 'blue'

//or like this
bird["food"]

Methods

A method is a function that is tied to an object. When you put in your names and press send it will call "person.setMessage()" then display your results.

Instantiations

To instantiate or initialize a new object you need to use the "new" keyword and pass in any needed parameters. In this case for the blueJay we don't need to pass anything in.
