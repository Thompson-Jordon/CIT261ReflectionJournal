JavaScript Events

Standard JavaScript Events

Here are a few event examples:

onclick

onclick fires when you 'click' on the element

onblur

onblur fires when you click on an element and then click off of it.

ontouch

ontouch is for mobile devices it fires when you touch the element on a touchscreen.

onmouseup

onmouseup fires when the mouse button is realeased.

mouse events

Im using 'mouseover' and 'mouseout' on this button to change the button color.
