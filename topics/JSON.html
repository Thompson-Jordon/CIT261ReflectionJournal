JSON

Introduction

JSON is a data type that allows for easy transfer of data between the server and the client. It's syntax is very similar to how you would create an object in JavaScript.

Parse

This is what JSON looks like in a .json file or as a string

{
  "ball: {
    "color": "green",
    "size": "small",
    "typeOf": "soccer"
  }
}

When you press this button we will parse this string into a json object.

We will now use the new object in a string.
