DOM Document Object Model

Introduction

The "DOM" was developed to make interacting between HTML and your programming language easier. In our case that is JavaScript but DOM works with other languages as well. DOM allows you to change, add, remove all of the HTML elements. It also allows you to change all of the CSS styles.
